# Pivots from `name`

A name is the most common and weakest seed — thousands share one. The whole game is disambiguation. Always pair the name with a discriminating fact (city, school, employer, age, relative) before searching.

## Sub-pivots
- **`social-profile/`** → [[pivot-name-to-accounts]] — search directory-style platforms and people-search engines, then disambiguate down to one person and harvest the confirmed profile.
